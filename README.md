# TZD Simple

Мінімальний офлайн-застосунок для сканування штрих-кодів EAN-13.

## Вимоги
- Android Studio Giraffe / Koala або новіша
- JDK 17+ (Android Studio в комплекті має рекомендовану версію)

## Як відкрити проєкт
1. Відкрийте Android Studio.
2. Оберіть **File → Get from VCS**.
3. Вставте URL цього репозиторію та натисніть **Clone**.
4. Після клонування дочекайтесь завершення синхронізації Gradle.

## Як протестувати сканер
1. Запустіть застосунок на фізичному пристрої.
2. На головному екрані натисніть кнопку **Scan / Сканувати**.
3. Надані права на камеру підтвердіть у діалоговому вікні.
4. Наведіть камеру на штрих-код EAN-13 на відстані 15–25 см.
5. Після успішного сканування екран закриється, а на головному екрані оновляться поля "Останній код" і "Всього зчитано".
6. За необхідності скористайтесь кнопкою ліхтарика на екрані сканування.
